# First Tornado App

Simple Tornado App


### Getting Started

1. Download [Python3](https://www.python.org/downloads/)

2. Clone Repository:

        $ git clone https://github.com/egarcia410/tornado-app.git

3. Change Directory:

        $ cd tornado-app

4. Install [Virtual Environment Wrapper](https://virtualenvwrapper.readthedocs.io/en/latest/install.html)

5. Create & Activate Virtual Environment

6. Install Tornado:

        $ pip3 install tornado

7. Run Program:

        $ python3 app.py
